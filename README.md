# SISTEMA DE CADASTRO DE COMPUTADORES

    Projeto em terminal feito para a segunda avaliação de OOP no terceiro semestre.

# Para rodar o sistema local:
     - Clone o repositório para sua máquina
     - Certifique-se de ter o JDK do JAVA instalado
     - Abra a pasta do projeto na sua IDE
     - Vá até o caminho './src/ui/Program.cs' e aperte RUN na sua IDE de escolha
